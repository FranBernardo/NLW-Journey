# NLW-Journey

## Setup instructions

### Clone repository
```
git clone https://github.com/FranBernardo/NLW-Journey.git
```

### Project setup

```
npm install
```

#### rodar em development

```
npm run dev
```

 ## technologies used

- Nodejs (linguagem que tenho mais famialidade)
- TypeScript (para fazer as tipagens)
- Prisma (como ORM)
